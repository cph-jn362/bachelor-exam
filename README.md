## Prerequisites 

1) Install the necessary dependencies (Both in the fronend & backend):

```bash
npm install
```
2) Add a new environment file in the root of the backend project:

```dotenv
POSTGRES_HOST=
POSTGRES_PORT=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DATABASE=
```

## Usage

Run development server

Frontend:

```bash
npm run dev
```

Backend:

```bash
npm run start:dev
```
